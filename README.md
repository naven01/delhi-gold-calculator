# Delhi Gold Calculator 🥇

A comprehensive gold price calculator for the Delhi region — built with historical data from 2000 to 2026, all-carat comparison, and price forecasting to 2035.

## Features
- **Historical pricing** — 2000 to 2026, annual averages for Delhi
- **All carats** — 14K, 18K, 22K, 24K side-by-side
- **What-if calculator** — "If I bought X grams in year Y, what is it worth today?"
- **Making charges & GST** — optional toggles for real purchase cost
- **Price forecast** — model estimates for 2027–2035 with confidence ranges
- **Year-by-year table** — full value history for your specific purchase
- **Interactive chart** — price, gain ₹, and return % views
